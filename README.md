# ForepaperEngineDemo
本项目为steam平台软件前景引擎ForepaperEngine的创意工坊内容开发示例，SteamKeys/Keys.txt中包含数量有限的兑换码，先到先得，商店地址在最下方。
This repository contains workshop sample projects for the ForepaperEngine software on the Steam platform. 
SteamKeys/Keys.txt contains keys for ForepaperEngine on Steam. These keys are for developers only and are limited in quantity, so please use them on a first-come, first-served basis. Steam store link:
https://store.steampowered.com/app/3601880?utm_source=github
